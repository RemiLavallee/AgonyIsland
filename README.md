# HorrorGame

Inventory system blueprint: https://www.youtube.com/watch?v=v8uq_20LFG0&t=4124s

The goal of the game is to escape the island with the boat.


INPUTS :

Move - W / A / S / D	

Jump - Space	

Look - Mouse 

Inspect - X

Exit Inspect - Right click

Interact - C

Crouch - Left Ctrl

Run - Left shift

Options - Escape

Inventory - I

Drop - O

Pick Up - F

Toggle Flashlight - N
